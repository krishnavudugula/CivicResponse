# CivicConnect

Civic issue reporting platform - Frontend prototype.

## What's working

- Home page with features
- Report submission form
- Clean UI/UX design
- Responsive layout

## What's NOT working yet

- Backend API
- Database (no data saves)
- File uploads
- User authentication

## Folder structure

```
CivicResponse/
├── pages/          # HTML files
├── css/            # Stylesheets
├── README.md
└── .gitignore
```

## How to run

Open `pages/home.html` in your browser or:

```bash
python -m http.server 8000
```

Then visit: `http://localhost:8000/pages/home.html`
